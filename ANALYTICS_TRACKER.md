## Sustainability Score

### Calculate Your Score:

```
Bandwidth:    ___% used → Score: ___/25
Functions:    ___% used → Score: ___/25
Duration:     ___% used → Score: ___/25
Performance:  ___/100   → Score: ___/25
                          ──────────────
              TOTAL SCORE: ___/100
```

**Scoring Guide:**
- 0-25% used = 25 points (excellent)
- 26-50% used = 20 points (good)
- 51-75% used = 15 points (moderate)
- 76-90% used = 10 points (concerning)
- 91%+ used = 5 points (critical)

**Overall Rating:**
- 90-100: Excellent sustainability
- 75-89: Good sustainability
- 60-74: Moderate - optimize soon
- Below 60: Action required
